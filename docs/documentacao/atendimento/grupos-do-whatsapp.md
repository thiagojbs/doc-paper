# Grupos do Whatsapp

Nesta seção, você encontra todos os guias necessários para criar e gerenciar conversas em grupos na plataforma. Aqui estão detalhadas as regras de criação para canais da API Oficial e Não Oficial, além de instruções completas sobre como utilizar as ações de controle Arquivar, Bloquear, Sair e Excluir em seus grupos.

## Guias Disponíveis

### [Criar Grupos (API Oficial)](./grupos-do-whatsapp/criar-grupos-api-oficial)
Aprenda como criar grupos utilizando canais da API Oficial do WhatsApp, seguindo as diretrizes e limitações da Meta.

### [Criar Grupos (API Não Oficial)](./grupos-do-whatsapp/criar-grupos-api-nao-oficial)
Veja o passo a passo para criar grupos através de canais da API Não Oficial (QR Code), com mais flexibilidade na gestão de participantes.

### [Gerenciar Grupos](./grupos-do-whatsapp/gerenciar-grupos)
Entenda como utilizar as ferramentas de gestão para Arquivar, Bloquear, Sair e Excluir grupos, mantendo sua lista de conversas organizada.
