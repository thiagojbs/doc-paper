# V.06

## Novidades dos Agentes de IA: Versão V.06

A versão V.06 permite, além do que já está previsto na versão v0.5 a orquestração de supervisores de agentes de IA;

### Habilidades

Aprimoramos como os agentes entendem as habilidades para evitar que ele envie o que está fazendo para o usuário final, por exemplo, "Adicionei a etiqueta Venda ao seu contato";

Habilidades internas como criar card no CRM, transferência e conclusão agora retornam para o agente a informação de sucesso e erro e não mais informações detalhadas do que foi executado. Assim ele não deixará exposto para o contato informações sensíveis.
