# Distribuição de atendimentos

**A funcionalidade de Distribuição de Atendimento encaminha conversas para os atendentes de uma equipe de forma sequencial, garantindo que cada conversa seja direcionada a um atendente por vez, de acordo com uma ordem predefinida.**

Isso assegura que as interações sejam distribuídas uniformemente entre os membros da equipe, otimizando o atendimento e evitando sobrecarga de trabalho em um único atendente.
