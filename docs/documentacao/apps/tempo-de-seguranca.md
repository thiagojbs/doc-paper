# Tempo de Segurança

O tempo de segurança na plataforma é uma funcionalidade que permite inserir um tempo de segurança de alguns segundos antes de enviar as mensagens aos contatos. Esse intervalo de tempo permite que você revise a mensagem e, se necessário, cancele o envio antes que ela seja enviada definitivamente.

::: tip Pré-requisitos
* Ter acesso à plataforma com perfil de administrador.
* Habilitar a funcionalidade na plataforma.
:::

## Para habilitar a funcionalidade

### Passo 1: Localizar o App

Na tela inicial, clique no menu **"Apps"** e localize o aplicativo **"Tempo de Segurança".**

![Localizar App Tempo de Segurança](/images/apps/tempo-de-seguranca/apps-tempo-seguranca.png)

### Passo 2: Habilitar

Localize o aplicativo "Tempo de Segurança" e clique em habilitar.

![Habilitar Tempo de Segurança](/images/apps/tempo-de-seguranca/habilitar-tempo.png)

### Passo 3: Configurar

Uma tela de configuração será aberta. Ative a chave para habilitar o aplicativo e defina o tempo de segurança, que pode ser de até 15 segundos. Finalmente, clique em "Salvar" para aplicar as configurações.

![Configurar Tempo de Segurança](/images/apps/tempo-de-seguranca/configurar-tempo.png)

## Veja em funcionamento

Se você enviou uma mensagem com um erro de digitação ou qualquer outro problema, pode cancelar o envio clicando em "Cancelar".

![Cancelar Envio](/images/apps/tempo-de-seguranca/cancelar-envio.png)

A mensagem será então marcada como "Mensagem apagada".

E o texto volta para a barra de digitação para ser editado ou apagado.

![Mensagem Apagada](/images/apps/tempo-de-seguranca/mensagem-apagada.png)

## Considerações finais

* A mensagem marcada como "Mensagem apagada" não é mostrada para o contato, ficará visível apenas para o usuário da plataforma.
* Após uma mensagem ser enviada não é possível apagá-la, isso é uma limitação da API Oficial da Meta.
