# Franquia e custo de mensagem

## O que é a API Oficial do WhatsApp?

A API oficial do WhatsApp é uma ferramenta desenvolvida pela Meta (anteriormente Facebook) que permite que empresas se comuniquem diretamente com seus clientes através do aplicativo WhatsApp. Diferente do WhatsApp Business tradicional, essa API é projetada para ser integrada em plataformas mais robustas.

## Franquia

A Meta oferece uma franquia ilimitada de mensagens gratuitas por mês para cada número de API. Essas mensagens gratuitas incluem mensagens receptivas, ou seja, mensagens recebidas do cliente para a empresa.

## Custo de Mensagem

Os custos variam dependendo do tipo de mensagem. Aqui estão os principais tipos de mensagens e seus custos:

* **Mensagens de Template Ativa (Modelos de mensagens do tipo Atendimento, campanha e sequências)**: São mensagens enviadas para abrir uma sessão de 24 horas. O custo varia entre $0,03 a $0,0625 por mensagem.
* **Mensagens Internas de Sessão**: São mensagens trocadas dentro da sessão de 24 horas. O custo é de $0,001 por mensagem. Há um teto máximo de $75 por conta, o que significa que, se o volume de mensagens for alto, a empresa pagará no máximo $75 por mês por conta, e o excedente fica ilimitado.

## Como Usar a API Oficial do WhatsApp

Para usar a API oficial, é necessário criar uma conta empresarial no Facebook e vincular o número de WhatsApp diretamente na Meta. Além disso, é preciso utilizar uma plataforma intermediária para gerenciar o número e facilitar o atendimento.

## Considerações Adicionais

* Os valores cobrados pela Meta são pagos diretamente a ela.
* No momento da vinculação do número à API oficial, é necessário cadastrar uma forma de pagamento utilizando um cartão de crédito internacional.
