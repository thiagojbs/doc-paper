# Modelo de Mensagens

Os **modelos de mensagem** são mensagens pré-aprovadas utilizadas para iniciar conversas no WhatsApp Business, garantindo comunicação eficiente e conforme as diretrizes da Meta. Eles são obrigatórios para **conversas de marketing, utilidade e autenticação**, permitindo o envio de notificações importantes, promoções e confirmações de identidade. Para serem utilizados, os modelos devem ser **aprovados pela Meta** antes do envio aos clientes. Além disso, é possível personalizar esses modelos com **variáveis dinâmicas**, tornando a comunicação mais personalizada e relevante.
